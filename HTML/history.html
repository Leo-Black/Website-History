Very Specific Facts About Python

By Leo Black

Home | History | Creator

The first version of Python was Python 0.9.0, published in February 1991. Later that month, version 0.9.1 was released, with some minor tweaks. However, version 0.9.2 included massive changes, making the language almost ready for a full release. The language gained several updates over the next couple of years, until version 1.0.0 was released on January 26th 1994. In October 2000, Python 2.0 came out. The next year, the non-profit orginisation Python Software Foundation (PSF). Demos for Python 3.0 began in August 2007. In September the next year, Python 3.0 was officially released. The update modified the software so much that it was no longer able to be backwards compatible. Because of this, Python 2.0 and Python 3.0 separated into two separate identities, with updates and improvements only coming to version 3.0.
